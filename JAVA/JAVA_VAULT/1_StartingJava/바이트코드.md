>**바이트코드**(Bytecode, portable code, p-code)는 특정 하드웨어가 아닌 [가상 컴퓨터](https://ko.wikipedia.org/wiki/%EA%B0%80%EC%83%81_%EC%BB%B4%ED%93%A8%ED%84%B0 "가상 컴퓨터")에서 돌아가는 [실행 프로그램](https://ko.wikipedia.org/wiki/%EC%BB%B4%ED%93%A8%ED%84%B0_%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4 "컴퓨터 소프트웨어")을 위한 [이진](https://ko.wikipedia.org/wiki/%EC%9D%B4%EC%A7%84%EB%B2%95 "이진법") 표현법이다. 하드웨어가 아닌 소프트웨어에 의해 처리되기 때문에, 보통 [기계어](https://ko.wikipedia.org/wiki/%EA%B8%B0%EA%B3%84%EC%96%B4 "기계어")보다 더 [추상적](https://ko.wikipedia.org/wiki/%EC%B6%94%EC%83%81%ED%99%94_(%EC%BB%B4%ED%93%A8%ED%84%B0_%EA%B3%BC%ED%95%99) "추상화 (컴퓨터 과학)")이다.

즉 자바 가상머신 위에서 돌아가게 하기 위한 중간코드이다! 이렇듯 기계어로 넘어가기 전 거치는 한 단계가 있기 때문에 C나 C++보다 느리고 인터프리터 언어보다 빠르다.
