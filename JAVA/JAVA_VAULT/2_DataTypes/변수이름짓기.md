### 변수명 작명 규칙
1. 숫자로 시작 불가
2. 특수문자는 $, _만 사용이 가능하다
3. 예약어 키워드는 변수명으로 사용이 불가하다.
### CamelCase
>변수 이름이 여러 어절로 되어있을 때 띄어쓰기를 하지 않고 각 단어의 첫 글자를 대문자로 작성하는 표기법, 데이터 타입의 범주에 따라 표기법이 조금씩 다르다.
1. 패키지 이름은 소문자로만
2. 클래스 이름은 첫문자가 대문자
3. 변수이름은 첫문자가 소문자

### 변수의 생존기한
>변수는 `선언`할 때 생겨나고 `닫는 괄호`에서 소멸한다.\


```java title:java
int a;  // 선언시점에 변수가 생성된다.
a = 10  // 할당과 선언을 구분할 것, 할당시점에는 생성 x
{
	int a = 33; // 할당과 선언을 동시에 할 경우 요때 생성된다.
} // 변수 b 소멸

```
