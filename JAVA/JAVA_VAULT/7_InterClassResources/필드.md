# 필드와 지역변수의 구분
---
* `필드`: 클래스에 포함된 변수
* 지역변수 : 메서드에 포함된 변수

* [[근데 어떤 변수가 어디 저장되는지 어떻게 암]]?

| 구분   | 포함  | 메모리 위치 |     |     |     |
| ---- | --- | ------ | --- | --- | --- |
| 필드   | 클래스 | 힙      |     |     |     |
| 지역변수 | 메소드 | 스택     |     |     |     |
|      |     |        |     |     |     |
|      |     |        |     |     |     |

# 필드와 지역변수의 초깃값
* 필드는 직접 초기화하지 않아도 강제로 초기화된다.
* 지역변수는 직접초기화 하지 않으면 빈 공간 그대로 있어서 값을 출력하고자 할때 오류가 발생한다.
