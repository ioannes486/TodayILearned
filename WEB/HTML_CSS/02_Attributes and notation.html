필기내용

마크태그는 형광펜, q태그는 따옴표 인용, blockquote는 노션인용같은 인용, i태그는 원래 이텔릭체인데 css를 사용해서 그냥 부분적으로 폰트를 바꾸는 용도로 사용한다.
cite태그는 인용문의 출처이름을 표기하는데 사용한다.
abbr태그는 약자를 표기한다.

인용문 관련 태그

blockquote와 cite 태그

HTML <blockquote> 요소는 안쪽의 텍스트가 긴 인용문임을 나타냅니다.
주로 들여쓰기를 한 것으로 그려집니다. (외형을 바꾸는 법은 사용 일람을 참고하세요)
인용문의 출처 URL은 cite 특성으로, 출처 텍스트는 <cite> 요소로 제공할 수 있습니다.

q태크에 대해 엠디엔 문서는 HTML <q>요소는 둘러싼 텍스트가 짧은
인라인 인용문이라는것을 나타냅니다. 라고 설명하고 있습니다.
