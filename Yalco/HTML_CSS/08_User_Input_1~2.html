필기내용

  • label의 for와 input의 id는 동일해야 한다. html에서id와 관련된 동작을 할 때 동일하게 작동한다.
  • button의 type속성들 (submit, reset : 초기화, button : js로 추가적인 기능을 구현할 때.)
  • form의 autocomplete속성 : 자동완성 on/off
  • fieldset의 disabled속성 : 중요한 내용을 다룰 때 사용, 입력창을 비활성화 시킨다. 속성 사용시 =이필요하지 않다.
  • input의 placeholder기능 : 빈칸에 어떤 내용을 적을 지 미리 알려주는 회색 텍스트를 지정한다.
  • search와 tel은 브라우저나 모바일 환경에 따라 다른 인풋형태를 보여주게 한다.
